"""
PhoneBookApp - простое приложение телефонной книги с графическим интерфейсом.

Функциональность:
- Добавление новой записи в телефонную книгу.
- Удаление существующей записи из телефонной книги.
- Поиск записей в телефонной книге по заданному запросу (подстроке).
- Просмотр детальной информации о выбранной записи (по ID записи).

Используемые технологии:
- Python для серверной и клиентской частей.
- Библиотека tkinter для графического интерфейса.
- Модуль threading для работы с многопоточностью.
- Модуль socket для обмена данными между клиентом и сервером по протоколу TCP.
- JSON для хранения данных телефонной книги в файле.

Структура приложения:
- gui.py - модуль с графическим интерфейсом пользователя.
- server.py - модуль с серверной частью приложения.
- client.py - модуль с клиентской частью приложения.
- main.py - для запуска графической и части серверной составляющей
- phone_book.json - файл для хранения данных телефонной книги.

Запуск приложения:
1. Запустите серверную часть приложения, выполнив start_server() из модуля server.py.
2. Запустите графический интерфейс, выполнив mainloop() из main.py.

ПРИМЕЧАНИЕ: связь с сервером по умолчанию происходит через адрес '127.0.0.1' и порт "12345"
	    Если у вас другой адрес хоста или данный порт недоступен, измените параметры host и порт в client.py и server.py (в функции start_server())

"""
